# 醫學資訊學系新生資訊 LINE Bot

## 專案概述

大一新生進入醫學資訊學系時，常對課程安排、師資背景、研究方向與系上活動感到陌生。然而過去系上缺乏統一且便利的即時資訊平台，導致新生在資訊取得上容易產生落差，增加適應新環境的困難。

為解決此問題，本專案開發了一款 **專為醫資系新生設計的 LINE Bot 聊天機器人**，提供 **24 小時即時資訊查詢服務**。透過簡單的對話與圖文選單，使用者可以快速取得系上相關資訊，降低資訊搜尋成本並提升新生適應效率。

在實際使用者測試中，系統獲得 **87.5% 的整體滿意度**。多數參與學生對於 **操作流暢度、資訊完整性、介面簡潔度與回應速度**皆給予正面回饋，顯示此工具能有效協助新生快速融入系上環境。



## 主要功能

本聊天機器人主畫面設計 **6 大核心圖文選單**，提供新生常用資訊：

### 關於醫資

* 師資名單與研究領域
* 教師辦公室位置與聯絡方式
* 系秘書聯絡資訊
* 畢業門檻說明

###  課程資訊

* 大一必修與選修課程介紹
* 授課教師資訊
* 教學方式與評分標準
* 參考書目

###  系學會活動

* 查詢近期系學會活動
* 活動時間與地點資訊

###  實驗室資訊

* 各教師實驗室介紹
* 研究方向與研究主題

###  繳費指引

* 繳費方式說明
* 官方繳費連結

###  意見回饋

* 內建 Google 表單回饋機制
* 蒐集使用者意見以持續優化系統



## 技術架構與使用工具
本專案透過 Webhook 串接 LINE Messaging API，並建置後端伺服器處理對話邏輯與訊息回應。

**開發語言**

* Python 3.12.5

**核心框架與套件**

* Flask 3.0.0
* line-bot-sdk 3.7

**開發環境**

* Visual Studio Code
* Windows 11

**測試工具**

* ngrok（建立本機臨時伺服器接收 LINE Webhook）

**部署與版本控制**

* GitHub（版本控制）
* Vercel（雲端部署）



## 系統運作流程

1. 使用者透過 LINE 發送訊息或點擊圖文選單
2. LINE Platform 透過 **Webhook** 將訊息傳送至後端伺服器
3. 部署於 **Vercel 的 Flask 應用程式**接收並解析訊息
4. 後端程式依據對話邏輯進行判斷與處理
5. 系統回傳對應資訊或互動選單給使用者

透過上述流程，系統可實現 **即時互動與資訊查詢服務**。
